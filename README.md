# 🚀 UrbanGo Smart Mobility

UrbanGo is a Full Stack Smart Mobility web application developed using PHP, MySQL, HTML, CSS, and JavaScript.

## 🌟 Features

- 🚌 Bus Ticket Booking
- 🚆 Train Ticket Booking
- 🚇 Metro Ticket Booking
- 🚖 Cab Booking
- ⚡ EV Charging Station Finder
- 💳 Secure Payment System
- 👤 User Login & Registration
- 🎫 Ticket Generation
- 📜 Booking History
- 📱 Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP

## 📌 Project Status

🚧 Currently under development with new features being added regularly.
